




* Magit
建立在 C-x g(打开界面) 之上
** 快捷键
s 对文件进行暂存(git add)操作
c 进入提交与历史管理

* 操作
** 分支相关
*** check out
查看Refs视图(显示所有分支) | y
跳转 | b b
新建并跳转 | b c
创建本地副本 | b l | 选择一个远程分支时，这个会创建一个本地的分支副本并切换过去
分离 | b s | 将在当前分支的提交 **剪切** 到新分支 并把当前分支恢复
*** create
新建分支但不切换 | b n
分离但不重置 | b S | 提交修改进行复制，而不是剪切到另一个分支
*** do 
配置 | b C
重命名  | b m | 将分支重命名
重置 | b x | 将分支强行切换到特定节点
删除 | b k | 删除分支

** 提交与历史
*** commit
标准提交 | c c | git commit
*** edit HEAD 
追加提交并修改信息 | c a | git commit --amend | 适用于漏了暂存文件并且提交信息补录
只追加,但不修改信息 | c e |
只修改信息,不追加 | c w |
*** edit
修补标记| c f | 创建一个新提交，标记为提交的一部分，并且不需要保留注释
压缩标记 | c s | 创建一个新提交，标记为提交的一部分，但是保留注释，会放在最后面
---- 以上两个有大写版本(F,S),需要配合 ll 选择历史提交一起使用，会把暂存文件直接合并到选中的历史提交中， F 会不会附加信息，S 会附加信息
